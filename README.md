## Getting Started

- Install all dependencies:


```bash
npm install
```

- Then, run the development server:

```bash
npm run dev
```

Finally, follow the CLI prompt(CTRL + click), and open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
